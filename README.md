Objective:

The objective of this program is to learn the fundamental concepts of C++ programming by storing and displaying student information using string variables and formatted console output. It also aims to develop an understanding of variable declaration, initialization, and the use of the cout statement to produce clear and organized output.

Outcome:

After completing this program, students will be able to:
Understand the basic structure of a C++ program.
Declare and initialize string variables.
Store and manage student information using variables.
Display formatted output using the cout statement.
Develop simple C++ programs that organize and present data effectively.
Gain confidence in applying basic programming concepts for future C++ applications.
